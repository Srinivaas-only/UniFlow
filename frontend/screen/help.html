Getting Started

1

Create an Account

Sign up with your email and password. Your data is securely stored in the cloud via Firebase.

2

Set Up Your Profile

Enter your name, university, and course. This helps the AI personalize responses for you.

3

Start Using AI Commands

Go to the AI Command Hub and type naturally: "calc quiz Thursday 2pm" or "spent RM15 on lunch". The AI will parse and organize everything.

4

Explore Your Dashboard

Check the Hub for an overview of upcoming events, budget stats, and quick actions. Navigate via the sidebar or bottom nav.

Yes! Your data is stored in Firebase Firestore with security rules that ensure only you can read/write your own data. Authentication is handled by Firebase Auth. Data is also cached locally in your browser for fast access, with background cloud sync.
Most features work offline since data is cached in localStorage. However, the AI Command Hub, Scholarship Finder, and Study Resources require the backend server to be running. Data syncs to the cloud when you're back online.
1. Install Python 3.8+
2. Run cd backend && pip install -r requirements.txt
3. Copy .env.example to .env and add your API keys
4. Run python -m uvicorn app.main:app --reload --port 8080

You need a DeepSeek or Groq API key for the AI parser. Scholarship/Resource search additionally needs a Bright Data API key.
